# Verkehrsampel

## Designkurzbeschreibung 

- 4 Seiten Ampel
- 1 Taster für An/Aus (WakeUp/Sleep) und Mode (Normal/Nacht)
- LEDs über Transistor angesteuert
- Interner Taktgeber

## Benutzung

Wenn Aus
* -> Taster drücken -> An

Wenn An 
* -> Taster kurz drücken -> Wechsel zwischen Tag/Nacht Modus
* -> Taster lang drücken -> Aus
 
## Verwendung 
- Ampel für Alma
